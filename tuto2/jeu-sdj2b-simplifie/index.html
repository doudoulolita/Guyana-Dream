Explications

Code inspiré des tutos "Créer un mini-RPG en JavaScript avec canvas" de Sébastien CAPARROS (sebcap26) et de Jamjam68140 qui a lancé le topic [big-tuto] Créer un rpg avec html5. Le code sera transformé davantage plus tard.

  • La balise canvas en HTML5 et javascript
  • Affichage d'une image sur le canvas
  • Affichage d'une partie d'image (tuile, sprite) sur le canvas
  • Affichage de parties différentes d'image sur la canvas, selon leur emplacement sur le tileset ou la spritesheet
  • Affichage d'une partie d'image sur une ligne horizontale
  • Affichage d'une partie d'image sur tout le canvas
  • Effacement d'une partie du canvas
  • Affichage de texte

Décors et personnages réalisés avec Blender à partir de Glitch et d'images libres de droit par Doudoulolita

Scripts: fichiers carte.js et feuille de style style.css.
