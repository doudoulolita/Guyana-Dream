01 - Colorer le canvas

Votre navigateur ne supporte pas HTML5!

Une fois la page HTML préparée avec la balise canvas, on va ajouter le code permettant de dessiner dessus.

La balise "script" permet de lier le fichier "script.js", rangé dans le dossier "js". C'est ce fichier qui permettra de travailler sur notre canvas à l'aide de javascript.


Fichier : index.html


	<!DOCTYPE html>
	<html>

		<head>	

			<title>Mon RPG perso</title>
			<meta charset="utf-8" />

			<link rel="stylesheet" type="text/css" href="css/style.css" /><!-- lien vers la feuille de style -->

			<!--[if IE]><script type="text/javascript" src="js/excanvas.js"></script><![endif]-->

		</head>

		<body>

			<h1>Mon RPG perso</h1><!-- titre de niveau 1 -->
	
			<canvas width="384" height="192">Votre navigateur ne supporte pas HTML5!</canvas><!-- emplacement du jeu -->

			<script type="text/javascript" src="js/jeu.js" ></script><!-- lien vers le fichier en javascript -->

		</body>
	</html>

Fichier : style.css, classé dans le dossier "css"


	canvas {
		border: solid; /* Ajoute une bordure au canvas */
	}

Fichier : jeu.js, classé dans le dossier "js"


		var canvas = document.querySelector('canvas'); //on récupère le canvas sur lequel on va dessiner dans une variable, grâce à la méthode querySelector
		var ctx1 = canvas.getContext('2d'); //on indique le contexte

		/* on remplit tout le canvas de couleur */
		ctx1.fillStyle="#5b6634"; // choix de la couleur (verte)
		ctx1.fillRect(0,0, 384, 384); // 0,0 sont les coordonnées x,y du coin supérieur haut du rectangle à remplir, les autres chiffre sont la largeur et la hauteur
		ctx1.fill(); // on remplit !
